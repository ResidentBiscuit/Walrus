# Walrus IRC
Modern IRC client built on top of [NW.js](https://github.com/nwjs/nw.js)

## Building
* Clone repository
* Install [NW.js](https://github.com/nwjs/nw.js)
* cd Walrus
* `npm install`
* `bower install`
* Add client.json to base directory (look at client.conf.example)
* `$PATH_TO_NWjs/nw .`
